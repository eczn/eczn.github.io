1.0 变量

JS变量很特殊 跟C的有较大差别。

这个差别主要体现在变量的值类型上面

1.1 基本数据类型

常见的，比如boolean型，number型都是基本数据类型

1.2 引用数据类型

比如一个对象：

var f = { hello: 'world' }

这个f里面存有后面那个“hello”的引用，即用 f.hello 可以得到 'world'
