两类铺平问题和它们的逆

这标题取的好数学 233 ，虽然如此，这些问题其实都遇到过，那么铺平是什么？

数组和对象都可以构成递归结构，暂且把使这类结构平铺成单个对象或者数组的过程称之为铺平，按被铺平的数据类型可以分为两类：

  1. 数组的铺平
  2. 对象的铺平

比如 [[1, 2], [3, 4]] 变成 [1, 2, 3, 4]
或者 {a: {b: 'hello'}, c: 'world'} 变成 ["a", "b", "c"] 或 ["hello", "world"]

    • typescriptinterface NestedArray<T> extends Array<T | NestedArray<T> > {}

function flat<T>(arr: NestedArray<T>): T[] {
    // 结果存在 result 
    const result: Array<T> = []; 
    // 我自己模拟一个栈
    const stack: NestedArray<T> = []; 

    while (true) {
        if (stack.length) {
            // 栈里有东西, 优先处理
            const list = stack.pop(); 

            if (Array.isArray(list)) {
                if (list.length) {
                    // 嗯 
                    const [firstOfList, ...rest] = list; 

                    if (Array.isArray(firstOfList)) {
                        // 第一个被压入了，第二个要截断，因为 firstOfList 是 list 的第一个 
                        stack.push(rest); 
                        stack.push(firstOfList); 
                    } else {
                        stack.push(rest); 
                        result.push(firstOfList); 
                    }
                } else {
                    // 遇到了空数组。。。 不管了 
                    // do nothing 
                }
            } else {
                result.push(list); 
            }
        } else {
            // 从 arr 里取出第一个元素出来。。。
            const first = arr.shift(); 
            
            if (typeof first === 'undefined') {
                // arr 没了，该结束了
                break; 
            } else {
                // 其余的情况扔进栈
                stack.push(first); 
            }
        }
    }

    return result; 
}

    # 另外一个角度解决对象平铺问题

    不论是 数组 还是 对象 都可以作为 JSON.stringify 的参数，并可以被转化成 JSON 字符串。

    由此，铺平问题就被转化成字符串问题了。

    调用处理刚刚的 hello : JSON.stringify(hello) 可以得到结果：

    "{"hello":"~","wolrd":{"nice":"~","to":"~","meet":"~","you":"~"}}"

    JSON中，在冒号前的是键名

    • jsvar flatJSON = o => {
    var str = JSON.stringify(o); 
    
    let tokens = ['{', '}', ':', ','].reduce((acc, e) => {
        // 给 '{' '}' ':' ',' 的两侧补上空格
        return replaceAll(acc, e, ` ${e} `);

        // 然后调用 split(' ') 把空格删掉，
        // 最后把空字符串滤掉
    }, str).split(' ').filter(e => e !== ""); 

    // 从 tokens 取得结果 
    // 冒号前的token就是键名 
    return tokens.reduce((acc, cur, idx, its) => {
        if (cur === ':'){
            acc.push(its[idx - 1]); 
        }
        
        return acc;
    }, []).map(e => cut(e, '"', ''));  
}
