    # ver α20170214 V2.5.0

    Qiniu Static Storage ADD
    简单的实现了 对md文件里面的图片的抓取 并保存到七牛云上

    主要做了这些事情

    • 增加 config.js 新项 qiniu, 里面有 域名、access_key、secret_key 等等
    • 新增 gulp generate: index.js 里 generator 最后阶段回调 vally.js 的 finish 以完成将 需要云存储的图片集 保存为 temp/list.dat
    • 修改 gulp deploy: 读取 list.dat, 跟 suc.dat 做比对，list里面存在的项如果在suc里也存在，则剔除，剔除后生成新集 need_to_upload ，最后把list和suc合并为新的 suc.dat，然后依次将 need_to_upload 上传到七牛

    # ver α20170214 V2.0.0

    Gulp Automatic ADD
    简单地做了下 Vally 的 Gulp 自动化

    V2 特性

    1. gulp serve // 启动vally服务器 供编辑使用
    2. gulp vally // 仅仅是编译出网站静态文件
    3. gulp deploy // 利用git工具部署到站点

    缺陷

    1. 不会自动创建 git 仓库
    2. 不会完全删除已存在的博文 （只在index.html里移除）
    3. gulp serve 存在异步bug ( reload 跟 generate 冲突 )
    4. 不会帮我把图片复制过去

    这是个很大的更新, 它意味着我可以偷更多的懒了, 此外这也同时意味着 Vally 进入的 αV2 阶段

    V3 完全体

    • 兼容以前的 V1 V2
    • Vally 能自己建好仓库，一次config，N次使用
    • 能将图片托管到七牛这样的静态存储网站
    • 将使用webpack， vally-loader 将成为 Vally 标志
