Hello, 2018

2017 年一晃眼的结束了，跟去年预料的一样，确实发生了很多事，是我最重要的一年。

# 编程喔

坚持编程，用 Git 管理代码。 今年写了很多很多 … 乱七八糟的代码 233

  1. Nep 计算机网络课设
  2. tplser 根据一些 PL 理论写的模板引擎
  3. interfacer 接口文档自动生成工具 （弃坑了）
  4. NUI 一个抄袭 Vue 的前端框架 （弃坑了）
  5. wx-area-chooser 小程序的一个小插件
  6. sftp-folder-upload 小工具，上传文件夹到服务器
  7. vue-ppp Vue 弹出层控制器
  8. nfs 模拟实现一个文件系统
  9. vally 继续各种改造、重构、尝试

可以看得见的功力提升

  1. 开始重视函数式了 (开始写些递归和几行实现 xxxx 了 2333)
  2. 习惯异步了 （。。。。写 JS 的都懂哈哈哈哈哈呜呜呜呜。。。）
  3. 大规模使用 ES6 （时代变迁）
  4. 性能虽然很重要，但是实现功能往往更重要 （吃了好多屎的经验）
  5. 今年的多个网络课程让我开始开始喜欢网络相关的理论了 （真爱）

# 被鄙视

不是说今年都是一帆风顺，有段时间很不爽 …

一是来自于工作室的，二是年中身体欠佳，还有就是我的学业

先说说一吧。

工作室这边，我自认为承受了太多，出于各种各样的原因我总是不得不来单干项目，或者帮别人擦屁股 …

刚开始觉得没什么，之后写多了，可能别人会产生习惯，之后就是无尽的 commit …

11 月许愿墙是真的累到我了，一方面早早起床去实习，一方面回城写码提交 … 唉，写码最繁忙的时候，陪着我的不是大三的叼毛，而是大二的几个儿子，真是伤到我。

也还好是那几个叼毛陪着我，在孤独感最强烈的时候

再加上我有是那种不太能发飙的人 … 所以催别人写码的时候根本没啥卵用 …

可能我也有错吧，翔翔年初跟我说的要背锅的人要暴力一些我这辈子是做不到了，哎，遗憾。

再来谈谈身体吧

感觉身体是不如刚刚读广工那会了，可以 2 点睡，8 点去上课还精神的很 … 最近上了班才知道好累呀，还要挤地铁，挤人流

年中的时候，持续的腰疼腰酸简直怀疑人生，去医院医生说没啥事，就是要多锻炼和休息 …

后来放了暑假才有时间休息，才恢复了，开学之后就没怎么敢修仙到四五点这样了。 （虽然有时也有）

至于学业嘛，我终于看清楚了：

大学考试的难度下的分数都是可以考前刷出来的，基本上不能代表个人能力，如果有，那就是考前突击能力。

考虑到这点后，我对以前的某些低分科目倒没啥遗憾了，毕竟该做的我都做了 233 （只是考试不小心失手了嘛 233 ）

还有就是，网络相关的理论虽然看起来很复杂，但实际上极其简单优雅 （尤其是网络层的包分组转发和路由机制），今年修了几门网络理论，感觉网络这块才是我的真爱 …

这才像是大学课程嘛 233

我又仔细的想了想，今年让我多休息多锻炼，那怎么可能嘛 …

今年真的是没时间 …

真的是没时间啊，一点个人的时间都没有，连续不断的输出、提交、120 加班

下半年的时候晚上、周六日、没课度假的时候基本上献给 120 了

后来单车报废了，常常一个人走在校园里，有段时间甚至有点享受

现在已经是 12 月了，也刚刚好这段时间静下来的时候才想到并且终于注意到了这破循环 …

想到我 6 月份我还想着能不能吧画画这项技能点起来 …
或者跑跑步减减肥锻炼锻炼这样 …

妈的这样来看，我还真是个大傻逼。

# 看星星

追到了最重要的人，开发了 desCode 传小情书 （笑）

也没啥了今年，没啥好说的了，我年年年底总结都会说，明年是我最重要的一年，明年应该也是我最重要的一年吧。

可是多多少少的那些遗憾、念想总喜欢在半夜打扰我，因为总是有失眠的时候。

好啦好啦，我都懒得扯了，随它去吧
