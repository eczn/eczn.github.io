    • jsjs-transform-01.jsconst res1 = [] + {}; 
// => '[object Object]' 

const res2 = {} + [];
// => '[object Object]' 

const res3 = eval('{} + []');
// => 0

console.log(res1);
console.log(res2);
console.log(res3);
    Run !

    # res1

    1. 计算 [] + {} 的时候，先要转换 [] 为原始类型，那该怎么转换呢，看加数的类型。
    2. 加数是对象，调用它的 valueOf, 结果还是对象，最后调用 toString() 得 ‘[object Object]’
    3. 之后计算 [] + ‘[object Object]’
    4. [] 调用 valueOf 得到它自己，然后调用 toString 得到空字符串，所以最终答案是 ‘[object Object]’

    • jsjs-transform-03.jsconsole.log(
    // string + object
    ('20' + {
        valueOf: () => 2, 
		toString: () => '20'
    })
);
// => '202'

console.log(
    // object + string
    ({
        valueOf: () => 2, 
		toString: () => '20'
    } + '20')
);
// => '220'

console.log(
    // object + object
    ({
        valueOf: () => 1, 
		toString: () => '10'
    } + {
        valueOf: () => 2, 
		toString: () => '20'
    })
);
// => 3
    Run !

    说明无论什么情况下，优先调用 valueOf，其次才是 toString，其调用顺序是操作数无关的，也就是说被加数或者加数是啥类型它不会管。
