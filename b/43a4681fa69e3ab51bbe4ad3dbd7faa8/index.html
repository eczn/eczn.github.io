# 整天沉迷学习 ↵

在协同的一个月里，睡的晚起得早，说实话我不太适应早起，平时就睡得晚，所以一个月里面迟到也不在少数，也有一口气迟到一上午的情况发生，在这里说声抱歉。
上课学到了很多啊，虽然好多没来得及吸收比如那个require，最近才知道这个东西用在node上比较多（引用包）。讲课讲的多是在一种比较大的范围或者仅仅讨论某种特殊情况来讲课，在课上学到的一般是比较笼统的认识，所以要在课后加强回顾（这点我一直都是厚颜无耻的）。
当然我是做不到一口气回顾暑假所学的全部东西的（好多东西都有些模糊），不过前端技术多种多样，变化非常快，学一个技术通常都是在某个项目需要的时候才去学（比如做主页的时候的bootstrap），最重要的是掌握这些库的一些核心思想，这才是前端学技术要把握的主线。
