# 电力猫和调制解调器 ↵

关于电力猫[1]猫在一般情况下指的就是 Modem，即调制解调器，是谐音；另外，以我个人的理解来看猫是一个将数字信号调变到模拟信号上进行传输，并解调收到的模拟信号以得到数字信号的电子设备（https://zh.wikipedia.org/wiki/%E8%B0%83%E5%88%B6%E8%A7%A3%E8%B0%83%E5%99%A8) (也算是一种隧道把 233) 网上已有很多科普文章，这里我将其总结为：电力猫实现了将以太网数字信号调制到交流电网的能力，相反它也具备将这样的信号从电力网络上还原[2]解调跟调制是一个相反的过程，将某种已被调制过的信号 “滤” 为初始的信号形式； 为以太网信号。

# 调制解调原理简述 ↵

大家都知道，交流电本身会不断交变，其电压随时间曲线满足形如以下的规则[3]交流电的函数关系这里我简单写成 sin(x) 了，实际上交流电还有些别的各种参数，这个请参考物理和电路课本，或者维基 https://zh.wikipedia.org/wiki/%E4%BA%A4%E6%B5%81%E9%9B%BB ：
AC = sin(x)
其中 AC 为交流电信号强度, x 为时间

# 电力猫结合阿苏斯 AiMesh 无线方案效果 ↵

挺好的，跑满百兆没问题，千兆目前还没跑到过。如果家庭里没有奇怪的电器的话，实际运行还挺稳定的，不会有啥特别的地方，但如果开了空调或者一些质量不太好的台灯总会有波动（变压器质量不好的那种） 还有就是不要插在抗电涌这类带滤波的插座上，信号会被屏蔽的
