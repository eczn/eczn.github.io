\n }\n}\n```\n\n示意图如下\n\n![作用域示意图](/images/scope-intro.svg)\n\n可以看到，上述的写法会产生两条单向链表： b-->a-->全局 和 d-->c-->全局 ，起点是 b 和 d。 \n\nb 执行的时候，需要搜索变量，其过程是：在当前作用域找，找不到就往上一级找，直到全局作用域也找不到就报错。 \n\n因为有两条链，很自然的，d 里面是不可能直接访问到 a 作用域内的 inner 的。 \n\n除非让 b 调用 d，让 b 吧 inner 给 d，换言之，通过传递闭包的方式来达到不同链之间的变量传递 ： \n\n``` js\nlet outer = 'outer';
