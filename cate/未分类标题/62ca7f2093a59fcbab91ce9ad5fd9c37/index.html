Markdown 快速上手
Markdown 是一种轻量级标记语言，创始人为约翰·格鲁伯，写文章经常用的到，它可以被编译成HTML，比起直接用HTML写文章，Markdown书写起来更简洁快速，写作体验会好很多。

# Markdown & HTML ↵

就我的日常而言，写出来的 Markdown 最终是编译成 HTML
我写博客也使用 Markdown 的方式书写，不过在某些方面已经进行了一些魔改

# 使用在线编辑器学习 ↵

Cmd Markdown 编辑器

# 基本语法 ↵ 

00# 顶级标题  对应  <h1>顶级标题</h1>
01
02## 二级标题  对应  <h2>二级标题</h2>
03
04### 三级标题  对应  <h3>三级标题</h3>
05
06> Text  对应  <blockquote>Text</blockquote>
07
08---  对应  <hr />
09
10**粗体**  对应  <strong>粗体</strong>  
11*斜体*  对应  <em>斜体</em>  
12`0x0`  对应  <code>0x0</code>
13
14# 图片
15![Alt text](/path/to/img.jpg)
16最后会编译为:
17<img src="/path/to/img.jpg" alt="Alt Text">
18
19# 链接
20[eczn's blog](https://eczn.github.io "blog")
21最后会编译为
22<a title="blog" href="https://eczn.github.io">
23    eczn's blog
24</a>
25
26需要注意的是 有些 Markdown 编辑器能混写 HTML 和 Markdown
