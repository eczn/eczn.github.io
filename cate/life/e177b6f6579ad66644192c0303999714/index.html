最后这段是写给大一的捞的，上面的绝大部分当笑话看就是了，明年你们也会写类似的或者更加羞耻的

而且，我有一种预感，未来的前端工程师很要做一些后台的工作，也就是说只有 Web 工程师了，这不是什么大势所趋，而是以 JavaScript 为核心的技术链条即将或者已经构成了覆盖前后端方方面面的一个闭包。
… 明年工作室的前端后台合并改成全栈组把。（误）
