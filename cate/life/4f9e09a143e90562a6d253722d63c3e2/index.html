又到年底了
今天周日，这里先给周什么程、Glory、 剑豪和一天说个对不起哈，我今晚其实是写这个叼毛玩意，许愿墙的总结几天前就写完了。
实在是想要说出来，有点受不了，日子越加阴暗，写出来、说出来我可能好受些 。。。。

# Grey ↵

这几个月，其实过的很快，我也没反应过来，就已经差不多到 12 月了。
开学的那会，在搞仲彬的项目，零零碎碎的学充实了 Vue，再来就是 Github 上瞎七瞎八搞的各种，挺有意思的，前段时间过了面试进了唯品做前端实习，感觉还不错，配 Mac 包 3 餐，只是从大学城过去有些麻烦，不过也还好，能克服，天天挤地铁，瞎想的时间也多了 …
这段时间搞了许愿墙，就项目情况而言，挺满意的，不过 … 我的心情，不算很好
这是一种拼尽全力的无力感，越努力越无力，许愿墙开发过程的来来去去，其实也就一个字： 累
可能曾有一段时间的幻想，幻想同伴会知道我的辛苦进而可能会过来大家一起拼全力去搞许愿墙、或者说，看到我挺努力的，会有所触动进而有个正向的过程。
然而事实上，没说出来的东西都是届不到的。
我太了解自己了，长久以来，我学不会说 不 吧，不论是高中被老师鄙视，或者，更加年轻的时候，跟其他人的交流 …
这会让别人产生一个严重的认知偏差：
  • 红莲做这些很厉害，继续让他做可能会更好
事实上，我从未有任何的编程天赋，都是我自找的。
我也向来反感吧什么都推到我身上，或者，做事情的时候别人在休息，而只有我一个人做。
人之常情。

# Blue ↵

前段时间思考这个问题，没有结论，只有怀疑，猜忌，愤怒 ———— 写代码的时候、只有我一人在开发的时候、 同伴出差错的时候
唉，可能是我太努力了，可是尽全力开发许愿墙这件事，不是理所当然的吗 ?

# Red ↵

其实没什么的只要退群就好了
既然都不能在一起协作，那又什么意义？
既然什么都靠我，我他妈是傻逼吗？
既然心情总是 bad bad 的，还呆在这里活受罪？
我从来都学不会正常的沟通 …
或许还有转机，因为总是挺开心的 … 最近还有油摩爽 …
少了这些，我可能都不算是完整的红莲了。
