# 站点样式 ↵

本站由 eczn/rally 强力驱动，唯一指定平台 Mac 或其他能很好的做衬线字体渲染的平台 (中文英文都要, 目前只有 Mac 做的最好)
本站写作跟普通的 Markdown 不同, 实现了一种动态执行渲染的机制, 在这种机制下, 文章内容是可编程的, 通过 React 表达的, 表现力比纯 Markdown 高的多, 随手就能在文章中写 Demo 跑想法。
